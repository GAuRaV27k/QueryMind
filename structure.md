# Project Structure and Module Connections

This file documents every folder and file in the repo and how they relate.

## Root

| Path | Details and Connections |
| --- | --- |
| `readme.txt` | High-level pipeline outline (conceptual flow). Not referenced by code. |
| `PROJECT_DEEP_ANALYSIS.md` | Design notes for the intended retrieval architecture. Not referenced by code. |
| `status.md` | Current implementation status and gaps. Documentation only. |
| `structure.md` | This structure map. Documentation only. |
| `.env` | Local secrets (API keys). Read by `API/*.py` clients via `dotenv`. |
| `.vscode/settings.json` | VS Code Python env defaults. No runtime impact. |
| `.claude/settings.local.json` | Local tool permissions. No runtime impact. |

## `API/` (External service clients and FastAPI entrypoint)

| Path | Details and Connections |
| --- | --- |
| `API/main.py` | FastAPI app with `/query`, `/multi_query`, `/step_back`. Imports `translate_chunk.multi_query` and `translate_chunk.step_back`. Does not call retrieval planning or providers. |
| `API/travily_client.py` | Loads Tavily API key from `.env`; exports `tavily_client`. Used by `search/provider/travily.py` and `retrieval/retrieval_manager.py`. |
| `API/you_client.py` | Loads You.com API key from `.env`; exports `you_client`. Used by `search/provider/you.py` and `retrieval/retrieval_manager.py`. |
| `API/exa_client.py` | Loads Exa API key from `.env`; exports `exa_client`. Used by `search/provider/exa.py` and `retrieval/retrieval_manager.py`. |
| `API/gemini_client.py` | Loads Gemini API key; exports `client`. Used by `translate_chunk/multi_query.py` and `translate_chunk/step_back.py`. |
| `API/groq_client.py` | Loads Groq API key; exports `client`. Used by `retrieval/retrieval_decision_maker.py`. |
| `API/Exa.txt` | Notes and examples for Exa responses/usage. Documentation only. |
| `API/__pycache__/exa_client.cpython-311.pyc` | Generated bytecode for `exa_client.py`. |
| `API/__pycache__/gemini_client.cpython-311.pyc` | Generated bytecode for `gemini_client.py`. |
| `API/__pycache__/groq_client.cpython-311.pyc` | Generated bytecode for `groq_client.py`. |
| `API/__pycache__/main.cpython-311.pyc` | Generated bytecode for `main.py`. |
| `API/__pycache__/travily_client.cpython-311.pyc` | Generated bytecode for `travily_client.py`. |
| `API/__pycache__/you.cpython-311.pyc` | Generated bytecode (likely from local `you` module usage). |
| `API/__pycache__/you_client.cpython-311.pyc` | Generated bytecode for `you_client.py`. |

## `translate_chunk/` (Query expansion via Gemini)

| Path | Details and Connections |
| --- | --- |
| `translate_chunk/multi_query.py` | Generates 4 paraphrased queries using Gemini. Called by `API/main.py` and `translate_chunk/extended_queries.py`. |
| `translate_chunk/step_back.py` | Generates 3 broader conceptual queries using Gemini. Called by `API/main.py` and `translate_chunk/extended_queries.py`. |
| `translate_chunk/extended_queries.py` | Query expansion entrypoint with provider fallback via `query_expansion/manager`. Used by `retrieval/retrieval_decision_maker.py`. |
| `translate_chunk/__pycache__/multi_query.cpython-311.pyc` | Generated bytecode for `multi_query.py`. |
| `translate_chunk/__pycache__/step_back.cpython-311.pyc` | Generated bytecode for `step_back.py`. |
| `translate_chunk/__pycache__/extended_queries.cpython-311.pyc` | Generated bytecode for `extended_queries.py`. |

## `query_expansion/` (Provider-managed query expansion)

| Path | Details and Connections |
| --- | --- |
| `query_expansion/types.py` | Provider interface, health tracking, retry policy, result metadata. |
| `query_expansion/utils.py` | Parsing, normalization, and retry classification helpers. |

## `retrieval/` (Planning, execution stubs, and normalization)

| Path | Details and Connections |
| --- | --- |
| `retrieval/retrieval_tool_capability.py` | Capability registry for providers. Used by `retrieval/retrieval_decision_maker.py`. |
| `retrieval/retrieval_plan.py` | `RetrievalPlan` dataclass. Used by `retrieval/retrieval_decision_maker.py`. |
| `retrieval/retrieval_decision_maker.py` | Groq-based planner. Imports `translate_chunk.extended_queries`, `retrieval_tool_capability`, `retrieval_plan`, `API.groq_client`. Produces plans but does not execute retrieval. |
| `retrieval/retrieval_decision_maker_doc.md` | Design doc describing the planner layer. Documentation only. |
| `retrieval/retrieval_manager.py` | Async provider calls for Tavily/You/Exa plus priority-based sizing. Imports API clients and provider wrappers. Not wired to the API layer. |
| `retrieval/Unified_retrieval_manager.py` | Normalizes provider responses into `UnifiedRetrievalResult`. Not invoked elsewhere yet. |
| `retrieval/__pycache__/retrieval_decision_maker.cpython-311.pyc` | Generated bytecode for `retrieval_decision_maker.py`. |
| `retrieval/__pycache__/retrieval_plan.cpython-311.pyc` | Generated bytecode for `retrieval_plan.py`. |
| `retrieval/__pycache__/retrieval_tool_capability.cpython-311.pyc` | Generated bytecode for `retrieval_tool_capability.py`. |
